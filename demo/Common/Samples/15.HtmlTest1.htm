简体中文

以著名的僵尸启示录为原型，加上一些具有新功能的新模组在 Minecraft 新版本中制作的丧尸围城整合包！旨在尽可能多地还原僵尸启示录。

开始

  • 建议使用超平坦模式，以村庄为根据地，建设防御工事（当然也可以游玩正常世界）

v1.x

  • 如果你想要更有趣和更具挑战性的游戏，可以在创建世界时点击数据包按钮并添加 Zombie Invade Nether End(僵尸入侵下界和末地) 和 Horde Nights(怪物之夜，即血月) 至右侧来加载它们

v2.x

  • 主世界默认生成失落城市（无需手动启用）
    • 正常超平坦世界无法生成失落城市，建议解决方法如下：创建世界时自定义超平坦 -> 预设 -> 主世界 -> 将 minecraft:bedrock,130*minecraft:dirt,minecraft:grass_block;minecraft:plains 替换至上方输入栏 -> 使用预设 -> 完成
    • 关闭主世界生成失落城市请前往 LostCities 模组设置 -> LostCities/common -> Profiles -> 删除 minecraft:overworld 这一项（点击右边按钮，提示文本错误仅为配置模组问题）

English

Remake from famous modpack Zombie Apocalypse, in new Minecraft with new features and mods! The pack aims to restore Zombie Apocalypse as much as possible.

Start

  • It's recommended to play in Superflat world type, building fortifications based on villages (of course you can still play normal world)

v1.x

  • If you want more interesting and challenging, you can click Data Packs button and add these datapacks when creating new world: Zombie Invade Nether End and Horde Nights

v2.x

  • The overworld now generates lost cities by default (no manually enable)
    • Normally, superflat worlds cannot generate lost cities, the suggested solution is as follows: Customise Superflat -> Presets -> Main world -> Replace minecraft:bedrock,130*minecraft:dirt,minecraft:grass_block;minecraft:plains in the input field above -> Use Preset -> Done
    • To disable the overworld generation of lost cities, please go to LostCities mod settings -> LostCities/common -> Profiles -> Delete minecraft:overworld (click the button on the right, the text error is just config mod's problem)

Tips

  • Server pack is included v1.2 and above (file named by server, bottom of the single version page)

    • Windows users run the startserver.bat file to install, then run again to start the server
    • For other systems, run the Forge installer forge-xxx-installer.jar directly (needs a Java that has been installed correctly),then run the script with start to start the server

  • The core mod of the modpack under v2.2 is Zombie Game (Thanks to the author Aljun2007 for giving me the idea to make this modpack), v2.2 is End of Days

  • The language of this pack was set to Chinese Simplified，you may set to others and disable/remove the following resourcepacks for Chinese translation. (if there is)

    • Jade 汉化.zip (v1.x)
    • SpartanWeaponry 汉化.zip (v1.x)
    • Xaeros地图+OPAC模组汉化包.zip (v1.x)
    • 夸克[Quark]配置汉化.zip (v1.x)
    • 真实物理[PhysicsMod]汉化.zip (v1.x)
    • ZI100D_补充汉化_txxx.zip (v2.x)
    • Minecraft-Mod-Language-Modpack-Converted-xxx.zip (all version)

  • Also disable/remove the following mods. (if you don't need)

    • IMBlocker (if you don't use an input method)
    • I18nUpdateMod
    • Just Enough Characters
